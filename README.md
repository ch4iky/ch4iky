Megan Geldenhuys - Professional CV Website
https://ch4iky.github.io/ch4iky/

A clean, modern, and interactive professional CV/portfolio website built with HTML, CSS, and JavaScript.
 Features

- Interactive tab system for Work Experience, Education, Projects, and Skills
- Rainbow animated header text
- Responsive design for desktop, tablet, and mobile
- Accessible with keyboard navigation
- Decorative animated stars with soft rounded edges
- Contact section with email, phone, and social links

## Technologies Used

- HTML5
- CSS3 (animations, gradients, responsive design)
- JavaScript (tab switching)
- Git for version control
- GitHub Pages for hosting

## Project Structure

ch4iky/
├── index.html          # Main HTML file
├── style.css           # All styles and animations
├── script.js           # Tab functionality
└── README.md           # Project documentation

## Installation

To run this project locally:

1. Clone the repository:
   git clone https://github.com/ch4iky/ch4iky.git

2. Navigate to the project folder:
   cd ch4iky

3. Open index.html in your browser
